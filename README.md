# Track-Viz

Roller Derby Track Visualization. Started out as a clone of [NURDS](https://nurds.space/).

- [Demo](https://trackviz.netlify.app/)
- [Docs](https://trackviz-docs.netlify.app/)

## Features

- 2D View
- Rotatable
- 3D View
- Import/Export
- Sequences
- Two different pack computation methods

## Requirements

Node 16+, npm

## Run

```bash
npm install
npm run dev
```
